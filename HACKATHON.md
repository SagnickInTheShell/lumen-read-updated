# Lumen Read - Hackathon Product Brief

## The Problem

Most people struggle while reading:
- Long pauses indicate confusion
- Scrolling back shows comprehension gaps  
- Slow speed reveals cognitive overload
- Tab switches reveal distraction

But reading apps don't adapt. They show the same interface to everyone.

## The Solution: Lumen Read

**"The reading app that notices struggle and adapts itself in real time."**

### Core Wow Factor: Adaptive Auto Mode ✨

The signature differentiator. When enabled, Lumen Read:

1. **Detects Struggle** in real-time:
   - Long pauses (10+ seconds)
   - Repeated scroll-back (re-reading)
   - Slow reading speed (<120 WPM)
   - Tab switching (distraction events)

2. **Adapts Automatically**:
   - 📏 Font spacing increases when reading slows down
   - 🌗 Contrast switches for visual comfort
   - 📖 Reading width narrows to reduce visual load
   - ✍️ Current line highlights for focus
   - 📝 Auto-simplification of difficult passages

3. **Measures Progress**:
   - Confidence Score (0-100%)
   - Focus Consistency tracking
   - Speed Recovery metrics
   - Distraction Reduction count

### Demo Experience Flow

```
1. User opens article
   ↓
2. App shows Adaptive Auto Mode toggle
   ↓
3. User enables it (magical moment - feels different)
   ↓
4. Dense paragraph appears → app detects pause
   ↓
5. TEXT TRANSFORMS LIVE:
   - Spacing increases
   - Font adjusts
   - Width narrows
   ↓
6. User continues reading smoothly
   ↓
7. Session ends → "You read 28% smoother today."
```

That's the WOW moment.

## Signature Features

### 1. **Adaptive Auto Mode** (The Headline)
- Real-time behavior detection
- Automatic UI adaptation
- No manual toggling needed
- Feels like the app is "reading" you

### 2. **Reading Confidence Score** (Emotional Impact)
- Shows measurable improvement
- Tracks focus, speed, completion, distractions
- Messages like: "You read 28% smoother today"
- Judges love quantified impact

### 3. **Onboarding Persona Selection**
Asks: "How do you struggle most while reading?"
- I lose focus
- Words blur together
- I read slowly
- I forget what I read
- English is hard

Then personalizes the entire UI. (Feels smarter than competitors)

### 4. **Premium Accessibility Modes**

#### Dyslexia Mode
- OpenDyslexic font
- 2.2x line height
- 0.08em letter spacing
- Warm pastel background (#fdf6e3)

#### ADHD Focus Tunnel
- One paragraph at a time
- Blur surrounding content
- Animated progress bar
- Reduced visual complexity

#### Low Vision Mode
- High contrast (dark background)
- 150%+ text zoom
- Yellow-on-black (#ffd700)
- Voice narration sync

### 5. **Silent Mode Coach** (Post-Session Feedback)
After reading session, shows personalized coaching:
- "You paused often in dense paragraphs. Try Focus Mode next time."
- "We noticed frequent distractions. Try full-screen mode."
- "Simplify Mode helped users like you read 40% faster."

Feels like an invisible tutor. Judges love this.

### 6. **Smart AI Micro-Actions** (No Generic Chatbot)
One-click inline actions:
- 📝 Simplify this paragraph
- 📖 Summarize section
- 💡 Explain difficult word
- 🌍 Translate sentence

Keep actions inside the reading flow. No separate chat panel.
